### 介绍

---
1. 天气客户端
2. 自定义：晴天、阴天、多云、雨天、雪天、雨夹雪、雾、霾、风、沙等天气动态背景
3. 利用和风天气api [https://dev.qweather.com/](https://dev.qweather.com/)感谢一波
