# Kotlin 版 天气客户端


## 介绍
---
完善自己开发的开源框架，丰富业余时间

## 简介

1. 天气客户端
2. 自定义：晴天、阴天、多云、雨天、雪天、雨夹雪、雾、霾、风、沙等天气动态背景

利采用 `MVP` + `okhttp3` + `RxJava` + `Retrofit` + `EventBus` 等架构设计的项目，项目代码结构清晰并且有详细注释 。

## 主要功能

- 根据高德获取当前城市，区域。
- 首页。获取指定城市的实时天气信息，24h天气状态，7天内天气状况，以及空气指数等等
- 城市管理。添加选中、搜索的城市，实时刷新所选城市天气
- 城市列表。热门城市列表。可搜索城市添加
- 天气模拟。模拟不同天气状况。
- 作者信息。
